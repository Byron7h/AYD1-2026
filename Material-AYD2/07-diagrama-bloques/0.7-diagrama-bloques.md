# Diagrama de Bloques

Diagrama de bloques del sistema que grita el negocio.

![Diagrama de Bloques](../imgs/DiagramaBloques.svg)

> El diagrama de bloques representa la arquitectura a nivel alto. Ayuda a mostrar como se agrupan modulos y como se comunican sin entrar en detalles UML. 

> Este diagrama es libre; puedes organizarlo como mejor represente tu arquitectura, incluyendo imágenes, para que alguien no tan relacionado con TI pueda entender el sistema.
