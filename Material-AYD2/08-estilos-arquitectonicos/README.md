# Seleccion del Estilo Arquitectonico

Esta seccion la verán más a profundidad en las clases magistrales de AYD2. El estilo elegido afecta como se organizan los componentes y el despliegue.

Estilos que se veran:
- Centrado en datos
- Por capas
- Flujo de datos
- Basado en eventos

> El estilo que elijan condicionara como organizan sus diagramas de componentes y despliegue.
