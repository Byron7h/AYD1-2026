# Casos de Uso Expandidos y Descripciones Textuales

Un caso de uso expandido detalla el flujo principal, alternativos y excepciones de un CU. La descripcion textual ayuda a transformar esos pasos en requerimientos funcionales.

---

> Se deben hacer los CU expandidos de todos los casos de uso de la primera descomposicion.
